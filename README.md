# vue-admin
## 项目说明
这是第一次尝试做一个后台管理项目，基于 Vue 的，希望借助这个项目，自己能对 Vue 全家桶、前端基础知识有个更全面的理解。同时也能做一些基础性的组件产出工作，便于以后自己复用。

可能会借鉴很多市面上已经做得比较完整优秀的后台管理框架，如有借鉴，也会在后面说明。

### 计划
- [] 栅格系统实现
- [] 框架结构搭建
- [] 皮肤样式切换
- [] mock Data
- [] dashboard
